# C++ Optimization Modules - Easy11 Backend

High-performance algorithms for search, caching, filtering, and recommendations.

## Modules

- **search.cpp**: Trie-based autocomplete
- **cache.cpp**: LRU cache
- **segment_tree.cpp**: Range queries
- **recommender.cpp**: ALS matrix factorization

## Build

```bash
node-gyp configure
node-gyp build
```

## Integration

Exposed via Node-API (N-API) bindings.
