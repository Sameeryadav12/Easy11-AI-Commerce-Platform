# Terraform Skeleton (Starter)

This folder contains a minimal Terraform starter for:
- VPC + subnets (multi-AZ)
- EKS (Kubernetes) or ECS Fargate
- RDS Postgres with replicas
- S3 buckets with versioning + replication
- CloudFront/Cloudflare fronting API/Admin/Vendor
- Secrets in AWS Secrets Manager + KMS keys

Note: This is a scaffold only; add providers, remote state, and per-env variables.
